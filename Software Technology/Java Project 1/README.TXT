PROJECT TITLE: Main Assignment (ST1)

PURPOSE OF PROJECT: To perform a couple of functions described on the 
assignment sheet.

VERSION or DATE: Finalised on 2/11/2018.

HOW TO START THIS PROJECT: Open this folder in BlueJ. Right-click on the 
class "Main" and select "void main(Strings[] args)".

AUTHORS: Rafi Rahman, Robert Cox

USER INSTRUCTIONS: When inputting letters, keep in mind that the program 
is case-sensitive. When inputting a text file name in the "QF" option, 
don't enter an invalid name, as the program will not be able to properly 
create the text file. You can only order up to 24 of each food product.
Have fun.
